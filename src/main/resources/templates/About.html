About

Created by Ian Jessop

Web Application that retrieves current and weekly forecast by using specified address or by using current geocode location.

Used Google Map Places API for the ability to autocomplete locations in form textbox.

Used Google Geocode API to check if entered location exists and to fetch location name and coordinates.

Retrieved current weather and weather forecast using Dark Sky API.

Technologies used: Spring Boot, Thymeleaf, Java, JavaScript, HTML, CSS
